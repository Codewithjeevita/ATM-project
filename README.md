# 🏦 ATM Project

A simple ATM (Automated Teller Machine) project built using Python.  
It simulates basic banking operations.

## 🚀 Features
- Deposit Money
- Withdraw Money
- Check Balance

## 🛠️ Technologies Used
- Python

## ▶️ How to Run
1. Open terminal
2. Run this command:
python ATM.py
